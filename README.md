## BLACKPANTHER

## Installation

Install blackpanther with git

```bash
git clone https://github.com/madhanmaaz/blackpanther
```
```bash
cd blackpanther
```
Install dependencies
```bash
npm install
```
```bash
pip install -r requirements.txt
```
